Step 01

자산 준비 및 링크 변환 Asset Prep & Link Conversion

GitHub에 올린 영상은 일반 링크로 바로 재생되지 않습니다. blob을 raw 주소로 변환하여 브라우저가 직접 파일을 읽을 수 있게 해야 합니다. Videos uploaded to GitHub won't play via standard links. You must convert blob to raw so the browser can read the file directly.

Pro Tip

"영상 파일명에 공백이 있다면 '%20'으로 자동 변환되니 주의하세요!" "Be careful, spaces in filenames are automatically converted to '%20'!"

Step 02

다크 모드와 UI 조화 Dark Mode & UI Harmony

최근 웹 트렌드는 다크 모드 지원이 필수입니다. Tailwind의 dark: 접두사를 사용하면 배경색과 텍스트 대비를 손쉽게 조절할 수 있습니다. Supporting dark mode is essential in modern web trends. Using Tailwind's dark: prefix makes it easy to adjust contrast.

Step 03

GitHub Pages 배포 GitHub Pages Deployment

Settings > Pages 메뉴에서 브랜치를 'main'으로 설정하세요. 수 분 내로 전 세계 어디서나 접속 가능한 당신만의 페이지가 완성됩니다. Set the branch to 'main' in Settings > Pages. In minutes, your page will be accessible globally.
